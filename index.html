Unit 2 Table - Intro to WebDev

4 Web Server Hacks This week I learned how to run 2 different types of local servers. One was a flask server and the other one was a docker server. I didn’t have any problems with the flask server, however I ran into some problems involving my M1 Macbook during the docker server.
